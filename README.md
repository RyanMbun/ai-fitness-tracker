# 🏋️ AI Fitness Tracker

A personal fitness tracking app I built with Python and Streamlit. I made this because I used to struggle with keeping a steady gym routine and knowing where to even start — this app helped make my gym journey easier and I wanted to build something that could do the same for others.

## What it does

- 📝 **Log your workouts** — enter the exercise, type, sets, reps, and weight
- 🏃 **Tracks cardio separately** — just input the exercise and duration
- 📊 **View your full workout history** in one place
- 🤖 **Generate a personalized AI workout plan** based on your goal, experience level, and how many days a week you can train

## Built with

- **Python**
- **Streamlit**
- **Google Gemini API**
- **JSON** for data storage

## How to run it

**1. Clone the repo**
**2. Install dependencies (pip install streamlit google-generativeai)**
**3. Set your Gemini API key ($env:GEMINI_API_KEY="your-key-here")**
**4. Run the app**
